## Design Direction

### Keywords

* Behance Style
* Pinterest Style
* Visual Impact
* Modern
* Clean
* Portfolio First

### Visual Style

* 화이트 베이스
* 강한 타이포그래피
* 대형 프로젝트 이미지
* 미니멀 UI
* 작품 중심 레이아웃

### Design Principles

* 방문자가 3초 안에 직무를 이해할 수 있어야 한다.
* 사이트보다 작업물이 먼저 보여야 한다.
* 시각적 임팩트를 제공한다.
* 모바일 우선으로 설계한다.

* 방문자가 3초 안에 직무를 이해한다.
